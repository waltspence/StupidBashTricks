# Stupid Bash Tricks
Just a set of useful useless bash scripts


[wxget](##Wxget)

Get the your or any locations weather in the terminal. Use the -l flag to specify the location you wish to get the weather for.

<strong>Examples</strong> 
```bash
wxget -l Austin
```
<strong> or </strong>

```bash
wxget -l 73301
```
